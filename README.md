A sample command-line application with an entrypoint in `bin/`, library code
in `lib/`, and example unit test in `test/`.

## Getting Started

Create a small app
------------------
```bash
$ dart create -t console <dirname>
```

Run the app
-----------
```bash
$ dart run
```

## Installing packages
Site: pub.dev

Install a package
-----------------
```bash
$ dart pub add <package>
```
